# BatchExport_AURAReport

日報のようなAURAレポートを指定期間分まとめてPDF出力するPowerShellスクリプトです。

# Features

* 日付をパラメータに持つレポートを指定期間分まとめてPDF形式でファイル出力できます。


# Demo

# Requirement

* ARIA AURA 
* PowerShell

# Installation
1. ```BatchExport_AURAReport.ps1```を用途に合わせて編集します。
2. PowerShellでスクリプトを実行します。

# Usage

* AURAレポートならなんでも
    * 日報
